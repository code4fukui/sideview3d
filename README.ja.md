# sideview3d
English README: [README.md](README.md)

シンプルなウェブベースの並列ビデオビューアです。

## デモ
デモは[GitHubリポジトリ](https://code4fukui.github.io/sideview3d/)で視聴できます。

## 機能
- 2つのビデオを横並びに表示します
- スライダーを使って2つのビデオの時間差を調整できます

## 使い方
1. `index.html` ファイルをウェブブラウザで開きます。
2. スライダーを使って2つのビデオの時間差を調整します。
